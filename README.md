# Smart Expense Tracker (Java)

A simple **console-based Java project** that helps users manage and analyze their daily expenses.  
This project uses **Object-Oriented Programming**, **Collections (Map, List)**, and **File Serialization** to store data permanently.  

## Features
-  Add expenses with **category, amount, date, and description**
-  View all past expenses neatly listed
-  See **total spending by category**
-  Saves all expenses even after program restarts (Serialization)
-  Demonstrates OOP concepts and Java Collections for real-world use

### Technologies Used
- **Language:** Java.  
- **Concepts:** Classes, Objects, Maps, File Handling, Serialization, Loops, Conditions.  
- **Tools:** VS Code / IntelliJ / Notepad + Command Line.
